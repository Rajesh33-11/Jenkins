
> **4. read** 

âœ” Best Use Case: 
  Splitting a line into multiple variables.

**Example**                   
name="teja rajesh kiran ganesh"                                 
age="35 25 63 25"                                                                              
place="bhimavaram goa hyd vizag"



```
#!/bin/bash
name="teja rajesh kiran ganesh"
age="35 25 63 25"
place="bhimavaram goa hyd vizag"

# Split values using read
read n1 n2 n3 n4 <<< "$name"
read a1 a2 a3 a4 <<< "$age"
read p1 p2 p3 p4 <<< "$place"

echo "hyy this is $n4 and I am from $p4 and my age is $a4"


````
